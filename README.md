# dome-takeout

### 项目介绍

一个外卖程序

该程序可以点外卖，结算金额，查看菜品,搜索地址，使用优惠券下单等

程序主要参照饿了吗，程序内的数据为虚拟数据

目前程序主要功能已完成

接口数据为随机，图片也为测试图片

### 页面布局

布局移动端布局 flex + rem
